# Speedport-API
Control Telekom Speedport routers with Python!
### Supported Devices
* Speedport Smart 4
### Installation
```commandline
pip install speedport-api
```
## Commandline tool
With this shipped commandline tool can a speedport in your network be controlled.
#### Turn wifi off
```commandline
speedport wifi off
```
#### Turn guest wifi on
```commandline
speedport guest-wifi on
```
#### Reconnect for new ip address
```commandline
speedport reconnect
```
#### Enable wps connect
```commandline
speedport wps
```
